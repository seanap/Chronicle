# Chronicle UI

This workspace is the Story 1.1 SPA scaffold using Vite + React + TypeScript + MUI.

## Commands

- `npm run dev` - Start Vite dev server.
- `npm run build` - Type-check and build production assets.
- `npm run preview` - Preview built assets.
- `npm run test` - Run Vitest suite.

## Integration Notes

- Dev proxy forwards API paths to Flask at `http://localhost:1609`.
- Backend endpoints remain source-of-truth and unchanged.
