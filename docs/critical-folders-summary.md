# Critical Folders Summary

- `chronicle/` — Core backend logic, API, worker, templates, stats
- `templates/` — HTML views for `/dashboard`, `/plan`, `/editor`, `/setup`, `/control`
- `static/` — Frontend JS/CSS assets
- `android/` — Android widget companion app
- `tests/` — Backend test suite
- `docs/` — Documentation and generated project docs
- `state/` — Runtime state, templates, and database (local)
- `local/` — Gitignored planning documents
